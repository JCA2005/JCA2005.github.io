El simulador de Lenovo es una herramienta web interactiva. Busca el modelo Legion 7-16ACHg6 (82N6) o un modelo equivalente si no aparece exactamente.

1) Abrir el simulador y seleccionar modelo

  1. Abre: https://download.lenovo.com/bsco/index.html#/
  2. Busca el modelo Legion 7-16ACHg6 o el código 82N6.
  3. Selecciona el equipo en la lista.

2) Entrar al menú principal de la UEFI/BIOS

  1. Haz clic en la pantalla del BIOS/UEFI.
  2. Localiza las pestañas: Main, Security, Boot, Exit.

3) Establecer contraseña de administrador

  1. En Security, busca Set Administrator Password.
  2. Introduce una contraseña robusta (mínimo 12 caracteres, mezcla de mayúsculas, minúsculas, números y símbolos): qkw0702ME(P>.

4) Establecer contraseña de arranque

  1. En Security o Boot, busca Set User Password, Power-On Password o HDD Password.
  2. Introduce la contraseña según tu política.

5) Deshabilitar arranque desde dispositivos externos

  1. En Boot, busca USB Boot.
  2. Cambia a Disabled.

6) Establecer orden de arranque seguro

  1. En Boot Mode, selecciona Windows Boot Manager como principal.
  2. Ponemos EFI PXE Network como segundo.

7) Habilitar Secure Boot

  1. En Security, busca Secure Boot y habilítalo.

8) Habilitar AMD Platform Security Processor

  1. Ve a Security y busca AMD Platform Security Processor y actívalo.

9) Desactivar Legacy/CSM

  1. En Boot, busca Legacy Boot y establece Disabled.
